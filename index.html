Frequently asked questions.


What is the software you use the most for Software developing?

I mostly use Microsoft Visual Studio and Visual Studio Code. But all other IDE's such as Brackets, Atom, Sublime text, etc are also good. But my favourites are Microsoft's Visual Studio and Visual Studio Code.


What is the best Game Engine?

Considering Game engines, The most used game engines are Unity and Unreal Engine 4. To be honest, I have only messed around with these two game engines. And as far as I consider, I would recommend Unreal Engine 4. It delivers the best graphics and easy coding by a unique feature provided in the engine called Blueprints. This game engine is used by most game developing companies. But the only flaw is that it takes up too much space on your PC due to these High Quality features.


Speaking of Unity. It is also one of the best Game engines out there. With it's latest updates, the game engine is reaching upto the level of Unreal Engine 4. The graphics provided by the game engine is improved by it's latest updates.The engine also provides some free game kits through it's asset store to developers to mess around and learn the structure and mechanisms. This engine is mostly used by Indie Game developers and also some Large Scale Game Developing Companies.


But these two engines cannot be compared as Good and Bad, because both the game engines are reaching out great heights due to quality and mechanisms.
